## 🚀 Como Acessar

1. **Acesse** a página de login do sistema
2. **Digite** o email do usuário desejado
3. **Digite** a senha correspondente
4. **Clique** em "Entrar"
5. **Será redirecionado** para o dashboard com sidebar específica do tipo de usuário
